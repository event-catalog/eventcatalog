## Overview

This section contains operational runbooks for all production services. Each runbook provides step-by-step procedures for common operational tasks and incident response.

## Available Runbooks

Browse the runbooks in this section to find procedures for:

- **Inventory Service** - Stock management and inventory sync issues
- **Orders Service** - Order processing and fulfillment problems
- **Payment Service** - Payment gateway and transaction failures
- **Shipping Service** - Shipping provider integrations and tracking

## Using Runbooks

1. Identify the affected service
2. Find the relevant runbook
3. Follow the diagnostic steps
4. Escalate if the issue persists
