The first Inventory Rebalancing Agent helped operations teams decide what to do when an order could not be fulfilled from the expected warehouse.

It received [[event|OutOfStock]] and recommended manual recovery actions before the later version added inventory-adjustment context and transfer planning.

## Responsibilities

- Explain which SKUs are constrained and where demand is building.
- Recommend transfer, substitution, or backorder actions.
- Give operations teams a short rationale for the proposed recovery path.
