Add OpenAPI spec import support to the DSL language server. Parses OpenAPI v3.0/v3.1 specs and maps GET operations to queries and POST/PUT/PATCH/DELETE to commands. Auto-detects AsyncAPI vs OpenAPI from file content. Supports local and remote imports, JSON and YAML formats. Includes playground examples for OpenAPI import, service import, remote import, and mixed AsyncAPI+OpenAPI usage.
