# EventCatalog for VS Code

Language support and live preview for [EventCatalog](https://eventcatalog.dev) DSL files (`.ec`).

## Features

- **Syntax highlighting** for `.ec` files
- **Live preview** — visualize your architecture as you type
- **Code completion** — autocomplete for DSL keywords and references
- **Diagnostics** — inline errors and warnings
- **Go to definition** — navigate between services, events, commands, and more

## Getting started

1. Install the extension from the [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=eventcatalog.eventcatalog-dsl)
2. Open or create a `.ec` file
3. Use the command **"EventCatalog: Open Preview"** (`Cmd+Shift+P` / `Ctrl+Shift+P`) to see a live visualization

## What is EventCatalog?

[EventCatalog](https://eventcatalog.dev) is an open-source documentation tool for Event-Driven Architectures. It helps teams document events, commands, queries, services, domains, and flows in a discoverable catalog.
