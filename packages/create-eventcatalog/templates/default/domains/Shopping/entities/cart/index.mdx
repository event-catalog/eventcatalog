    referencesIdentifier: customerId

    type: string
    required: true

    type: datetime
    required: true
    description: Time the cart last changed.

## Overview

Cart is the Shopping domain aggregate. It accepts item changes and emits [[event|cart-checked-out]] when the customer commits to buy.
