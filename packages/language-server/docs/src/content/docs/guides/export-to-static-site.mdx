Export your `.ec` files as standalone static diagrams and pages. Share architecture visualizations without requiring anyone to install tooling or run a server.

## Planned capabilities

- Export `.ec` files as static HTML pages with interactive diagrams
- Generate PNG or SVG images of your architecture
- Share a link to a self-contained page that anyone can view in a browser
- Embed diagrams in wikis, READMEs, or slide decks
