### `main.ec`

```ec title="main.ec"
visualizer main {
  service UserService {
    version 1.0.0
    summary "Manages user accounts"

    sends event UserCreated
    sends event UserUpdated

    receives command CreateUser
    receives command UpdateUser
    receives query GetUser
  }
}

```
