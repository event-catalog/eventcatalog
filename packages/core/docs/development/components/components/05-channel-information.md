The ChannelInformation component is a EventCatalog component that will render a table into your markdown page that contains information about the Channel. 

**Example**

```jsx /services/MyService/index.mdx
<ChannelInformation />
```

### Output
![Example output](./img/channel.png)

See example in the [demo EventCatalog application](https://demo.eventcatalog.dev/docs/channels/inventory.%7Benv%7D.events/1.0.0).

### Support

The `<ChannelInformation/>` component is supported in channels.
